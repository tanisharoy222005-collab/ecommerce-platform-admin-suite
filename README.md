# E-Commerce Platform with Admin Suite

Production-ready e-commerce platform showcasing:

- Product Catalog
- Shopping Cart
- Checkout Flow
- Admin Dashboard
- Order Management

## Technology Stack

- Next.js
- React
- Vercel

## Deployment

1. Upload repository to GitHub
2. Connect GitHub to Vercel
3. Deploy automatically

## Features

- Responsive design
- Product listing
- Admin dashboard
- Checkout workflow
- Order management interface
